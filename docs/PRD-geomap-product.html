1. Visió del Producte

"La primera plataforma que permet saber, en temps real i per qualsevol parcel·la agrícola de Catalunya, si un camp està cultivat o abandonat — i per quant de temps."

Creiem que propietaris, inversors, asseguradores i administracions necessiten dades fiables sobre l'estat real dels camps agrícoles. Avui aquesta informació no existeix de forma accessible, actualitzada ni estructurada. Nosaltres la fem possible creuant dades catastrals amb imatges de satèl·lit Sentinel-2 (actualitzades cada 5 dies).

6. Roadmap de Producte

🟢 Fase 1 — MVP Web (2 mesos)

  • Landing page pública (Next.js)
  • Integració mapa actual
  • Registre / Login (email + Google)
  • Fitxa parcel·la (/parcela/:ref)
  • Rate limiting per usuari (Redis)
  • Deploy a producció (Railway)
  • Integració Stripe (subscripcions)
  • Pàgina de preus

🟡 Fase 2 — Monetització (1 mes)

  • Portal de client (gestió pla)
  • Generació PDF per parcel·la
  • Sistema d'alertes per email
  • API keys per Professional/Enterprise
  • Dashboard analítics bàsic
  • Onboarding flow (tour guiat)

🔵 Fase 3 — Creixement (3 mesos)

  • Cobertura a tota Espanya
  • Comparativa temporal NDVI
  • Integració SIGPAC (ús declarat vs real)
  • Widget embeddable tercers
  • SEO: pàgines per municipi
  • App mòbil (PWA)
